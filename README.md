豆包 关于波音737起飞临界速度的回复： 波音737的起飞临界速度（默认指运行层面的起飞安全速度V2，即单发失效后仍能保持稳定安全爬升的最低临界速度）不是固定值， 随机型、业载、机场海拔、气温、跑道条件波动，典型场景数值如下：

标准条件（海平面、15℃、干跑道）下的典型值：
737经典系列（-300/-400/-500）：最大起飞重量下为130155节（241287km/h）
737NG系列（最常见的-700/-800）：最大起飞重量下为135160节（250296km/h）;150座级中等载客、半油场景下为130145节（241269km/h）
737MAX系列：最大起飞重量下为140165节（259306km/h）
特殊场景波动：
高原高温机场（海拔3000m以上、气温30°C以上）：比标准条件高1525节（2846km/h）
低业载短航程场景：可低至115125节（213232km/h）
若指起飞构型下的气动失速临界速度，737全系列标准条件下约110125节（204232km/h），实际运行中V2会比该值高20%以上，不会触及失速临界。

注意事项：代码中的API Key要为真实的个人获取的密钥，Python版本要大于3.4
